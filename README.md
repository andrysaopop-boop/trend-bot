# Trend Bot

A simple Python tool for analyzing crypto market trend direction.

## Features

- Fetches OHLCV market data
- Calculates EMA20, EMA50, EMA200
- Calculates RSI
- Calculates MACD
- Calculates ADX
- Finds support and resistance
- Gives a trend summary: uptrend, downtrend, sideways

## Example assets

- SOL/USDT
- BTC/USDT
- SUI/USDT

## Purpose

This project is built for learning market structure, indicators, and basic systematic analysis.
